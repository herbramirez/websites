3D Design

Works done using either Maya, Cinema4d, Marvelous Designer, Cl03D, and Substance Designer

FORM

Software: Maya

Using Maya and base models from Fusion I wanted to recreate bodies forming and deforming using procedural effects. The animation can also be viewed using a VR using VRplugin for Maya. The meshes can be exported and readily available for use in Unity for both virtual reality and augmented reality.

Garment

Software: Maya, Marvelous Designer, Clo3D

Using real life fashion patterns designed by Open Style Labs, I created a 3D version of their jacket and animated it using various softwares.
